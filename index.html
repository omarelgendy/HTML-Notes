My HTML Notes


Terms

  1. Markup: Text with special meaning that tells the browser how to display text Ex: colour, font and size
  2. Void Tags: Tags that don't have a closing tag as they have no content
  3. Nesting: Putting bits of code inside other bits of code in order
  4. Block Elements: Elements that creates a block containing the data inside them
  5. Inline Elements: Elements that just exist and are flowing in the page
  6. Don't need to close: The browser knows that these types of elements can't exist inside each other so it automatically closes the first one
  7. Qualified URL: A URL which has the full path leading to the file or that contains protocol, Domain Name and directory
  8. HTML Attribute: An extra piece of data that comes with a HTML code to give in extra meaning

Tags

  1. p: Creates a paragraph of its components (Don't need to close)
  2. br: Line Break (No closing)
  3. mark: Highlight the text
  4. sup: Moves text up (as powers in math)
  5. sub: Moves the text down (Reverse of sup)
  6. em: Display text as italic
  7. strong: Display the text as bold
  8. h1: Main page heading
  9. h2 till h6: Smaller titles in the page
  10. div: Divides the page into sections
  11. ol: Creates an ordered list bulleted by numbers
  12. ul: Creates an unordered lists bulleted by dots
  13. li: Creates an element in the list & can't exist without being nested in ol or ul (Don't need to close)
  14. a: Creates a hyperlink and usually comes with the href Attribute
  15. img: Adds an image and always comes with the src(source) and the alt(alternate) attibutes
  16. link rel="stylesheet" href="URL": links css if written in a separate sheet
